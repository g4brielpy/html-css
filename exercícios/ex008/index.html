Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em Negrito usando a tag < b > não semâtica (somente formula, sem significado).
Nesta frase temos um termo em Destaque usando o < strong >, semâtica (Tem significado, dar destaque a um termo)

Itálico / Ênfase

Nesta frase termos um Termo em Itálico usando a tag < i >, não semâtica (somente forma, sem significado)
Nesta frase termos em ênfase usando a tag < em >, semâtica (Tem significado, indica um termo em ênfase)

Texto Marcado

Podemos marcar texto usando a tag a < MARK >

E no outro parágrafo, a tag < mark > fica na cor configurada no head

Texto Grande / Pequeno

Podemos criar textos grande usando a tag < big >, não semâtica(tag obsoleta)

E podemos criar textos pequenos usando a tag < small >, semâtica (texto secundário, mas importante)

Texto Deletado

Podemos marcar um texto como ixcluído que pode ser lido, mas desconsiderado. usando a tag < del >, (semâtica)

Texto Inserido

Podemos marcar um texto Inserido usando a tag < ins >,semâtica (Indicando que o texto foi Inserido)

Existe também o sublinhado com a tag < u >, não semâtica

Texto Sobrescrito

para inserir coisas do tipo x20+3, tag < sup >, (semâtica)

Texto subscrito

Para inserir coisas do tipo H2O, tag < sub >, (semâtica)
